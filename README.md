## Getting Started

```bash
pnpm i
```

## Smart Contract Testing

Navigate to the moda-contracts directory and run

```bash
pnpm test
```

To generate a coverage report run

```bash
pnpm coverage
```

## Smart Contract Documentation

1. `pnpm contracts:docs`
2. Visit [localhost:3000](localhost:3000)
